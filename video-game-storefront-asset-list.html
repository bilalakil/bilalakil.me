Video Game Storefront Asset List

Last updated: 2022/02/12

iOS

Sources
  • Screenshot specs
  • App preview / trailer specs

Mandatory

  • App icon: 1024 x 1024px resolution
  • Screenshots. 1 of each of the following resolutions are mandatory; any more are optional (limited to 10 each):
    • 2778 x 1284 or 2688 x 1242 (or switch for portrait)
    • 2208 x 1242 (or switch for portrait)
    • 2732 x 2048 (or switch for portrait)

Optional

  • App preview / trailers. Limited to 3 of each. Must be 30 frames per second, between 15-30 seconds long, and of these resolutions:
    • To show up on 6.5 / 5.8 inch iPhone App Stores: 1920 x 886 (or switch for portrait)
    • To show up on all other iPhone App Stores: 1920 x 1080 (or switch for portrait)
    • To show up on iPad App Stores: 1600 x 1200 (or switch for portrait)
